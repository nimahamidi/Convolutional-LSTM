# LSTM Pose Machines in Keras

Implementation of LSTM Pose Machines with Keras

This is keras implement of [LSTM Pose Machines](https://arxiv.org/pdf/1712.06316.pdf)


# Prerequisites
- Python 3
- keras
- tensorflow
- openCV
- pandas
- numpy

# Train
python lstm_pm_train.py

# Test
python test_lstm_pm.py
